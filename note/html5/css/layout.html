top/right/bottom/left

同时设置top和bottom或同时设置left和right则会拉伸元素

top/left/right/bottom 指的是元素到对应边的距离


z-index

z-index存在栈的概念,不是z-index大的一定在上面,首先要看其父元素的z-index,再比较子元素的z-index


position 定位

  • relative|absolute|fixed

relative 相对定位(常用于绝对定位的参照物)

  • 仍在文档流中

  • 参照物为元素本身

absolute

  • 默认宽度为内容宽度

  • 脱离文档流

  • 参照物为第一个定位祖先/根元素

fixed

  • 默认宽度为内容宽度

  • 脱离文档流

  • 参照物为视窗

flex 弹性布局

创建flex container

  • display:flex

弹性元素 flex item

  • 弹性元素指的是容器的文档流中的子元素（不包括孙元素等）

flex-direction 方向

  • flex-direction:row|row-reverse|column|column-reverse

flex-wrap 换行

  • flex-wrap:nowrap|wrap|wrap-reverse

flex-flow 流（推荐使用）

  • flex-flow:<'flex-direction'>||<'flex-wrap'>

order排序

  • order:<integer>

    • order的值大的排在后面


弹性

flex-basis 基础宽度

  • flex-basis:main-size|<width>

flex-grow 元素分配到剩余空间的比例

  • flex-grow<number>

flxe-shrink 分配多出的空间

  • flex-shrink<number>

flex 缩写

  • flex:<'flex-grow'>||<'flex-shrink'>||<'flex-basis'>


对齐

justify-content 主轴对齐方式

  • justify-content:flex-start|flex-end|center|space-between|space-around

align-items 辅轴对齐方式

  • align-items:flex-start|flex-end|center|baseline|stretch

align-self 单个元素对齐

  • align-self:auto|flex-start|flex-end|center|baseline|stretch

align-content 多行对齐

  • align-content:flex-start|flex-end|center|space-between|space-around|stretch
